HTML

Ideas principales

  • La gran mayoría de las etiquetas deben tener un inicio y un fin.
  • El inicio de una etiqueta es normal, en otras palabras lleva los dos caracteres y dentro de estos se encuentra el nombre.
  • Para el cierre de la etiqueta antes de escribir el nombre de la etiqueta debemos escribir una diagonal /
  • Entre el inicio y el fin de la etiqueta debe estar el contenido.

Apuntes

  • Las etiquetas en su mayoria tienen un cierre, pero hay excepciones, algunas no tienen cierre.
  • Cuando una etiqueta este incompleta el signo de "Menor que" se tornara color rojo.
  • El contenido tiene que estar entre el "Mayor que" de la apertura y el "Menor que" del cierre.

Resumen

En resumen podemos establecer que html tiene un lenguaje estricto, pero muy noble y nos ayuda a completar las etiquetas facilmente sin errores con la funcion "Abreviacion Emmet".

Texto en HTML

Ideas principales

  • • La mayor parte del texto estructurado está compuesto por encabezados y párrafos, independientemente de si lees una historia, un periódico, un libro de texto, una revista, etc.
  • • El contenido estructurado simplifica la experiencia en la lectura y se disfruta más.
  • • Una de las principales funciones de HTML es dar al texto estructura y significado de forma que un navegador pueda mostrarlo correctamente

Apuntes

  • "Lorem" para poner parrafos de texto aleatorio.
  • Existen las listas ordenadas (ol) y las listas desordenadas (ul), su diferencia es que la ordenada tiene un orden establecido por numeros y la desordenada no lo tiene.
  • Hay manera de establecer estilos a la tipografia mediante comandos como "i" que aplica estilos a la letra italika (em)
  • Existen los h, que le dan jerarquia al texto, el h1 siendo el tamaño más grande de tipografía, el h2 siendo un subtitulo y llegando hasta h6 siendo el más pequeño de todos

Resumen

En resumen, el texto tiene muchas herramientas, subrayado, negrita, estilos diferentes, creacion de parrafos, titulos y demás, es simple de aplicar

Imagenes en HTML

Ideas principales

  • Para añadir imágenes a un proyecto web hay que escribir el siguiente código HTML: Atributo src="".
  • El atributo src="" indica la ruta de la imagen que se quiere mostrar.
  • img src="archivo.jpg".
  • Si la imagen es insertada desde una pagina web la ruta se escribe: img src="http://web.com/carpeta/archivo.jpg"

Apuntes

  • La etiqueta img no tiene cierre.
  • Las imagenes pueden ser manipuladas mediante width (Ancho) o height(Altura), de preferencia solo modificar 1 atributo para que se reescale automático
  • Nunca estirar demás las imagenes o hacer que se vean deformadas, es preferible manipularlas en photoshop.

Resumen

En resumen podemos decir que es muy facil incorporar imagenes y manipularlas en html y css, simplemente hay que tener en orden nuestra carpeta del proyecto web, para que las rutas de archivos funcionen correctamente.

Enlaces en HTML

Ideas principales

  • El atributo target especifica dónde abrir el documento vinculado. puede tener los siguientes valores:
  • _self: predeterminado. Abre el documento en la misma ventana / pestaña en la que se hizo clic
  • _blank - Abre el documento en una nueva ventana o pestaña
  • top: abre el documento en el cuerpo completo de la ventana
  • Links de imagen:Podemos usar una imagen como link, simplemente poniendo img dentro de la etiqueta a
  • Ejemplo: a href="#">img src="">/a>

Apuntes

  • Los enlaces facilitan mucho el trabajo y la experiencia del usuario en la navegación
  • Son faciles de aplicar y usan la etiqueta "a"

Resumen

Podemos decir que los enlaces, funcionan como puentes hacia funciones simples, ayudan a la navegacion y a las rutas de archivos.

CSS

Ideas principales

  • Las hojas de estilo en cascada o CSS (Cascading Style Sheets) es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado (por ejemplo, HTML).
  • Son un mecanismo simple para agregar estilo o diseño por ejemplo, fuentes, colores, espaciado a documentos web.
  • A los elementos en general de la pagina les proporciona apariencia deseada, mediante formas, margenenes, colores etc.

Apuntes

  • El css va ligado al html en el head del html.
  • Trabaja con clases en cada elemento despues de asignarle un nombre o identificador
  • Para manipular las clases en css se tiene que escribir:
  • header{
  • width: 1366px;
  • height: 250px;
  • background-color: antiquewhite;
  • text-align: center;
  • background-image: url(img/Fondo\ comics.jpg);
  • margin: 0 auto;
  • }

Resumen

El css o hoja de estilo en cascada, es la manera de personalizar cada pagina web, elemento por elemento, dando miles de opciones para darle una vista agradable y estética.

Proceso de diseño

Ideas principales

  • El proceso de diseño, es el proceso que involucra una indagacion o investigacion previa, para estableces los puntos importantes, y diseñar de manera eficaz.
  • Se enfoca en diseñar en base a un usuario o clientes
  • Sigue una serie de pasos bien fundamentados por informacion veridica, modela al usuario y garantiza el exito del producto.

Apuntes

  • Indispensable modelar al usuario.
  • Se enfoca en solucionar un problema y satisfacer una necesidad.
  • Sigue una serie de pasos y tiene que cumplir ciertos requisitos.

Resumen

El proceso de diseño es un proceso indispensable para poder crear y diseñar las cosas eficazmente y asi obtener los resultados deseados.
